# LEGO MINDSTORMS EV3 Saved Programs

A collection of saved LEGO MINDSTORMS EV3 programs (`.lmsp` files) to share with the community.

## Files

- `Robot Arm H25.lmsp` — Program for the Robot Arm H25 model from the LEGO Education core models curriculum

## How to use

1. Download and install the [LEGO MINDSTORMS Education EV3 software](https://education.lego.com/en-us/downloads/mindstorms-ev3/software)
2. Open any `.lmsp` file from this repository in the EV3 software
3. Connect your EV3 brick via Bluetooth or USB and download the program

## File format

`.lmsp` (LEGO MINDSTORMS Project) files are zip archives containing a `manifest.json` (project metadata), `scratch.sb3` (the Scratch-based block program), and `icon.svg` (project icon).
